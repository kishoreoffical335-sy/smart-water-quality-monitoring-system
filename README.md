## 📌 Project Overview

The **Smart Water Quality Monitoring System** is a Machine Learning-based web application that predicts **Dissolved Oxygen (DO)** levels and determines whether water is **SAFE, WARNING, or UNSAFE**.

This system helps in analyzing water quality quickly using key parameters like:
- pH
- Turbidity
- Temperature

Machine learning automates water quality assessment, reducing manual effort and improving efficiency. :contentReference[oaicite:0]{index=0}

---

## 🎯 Features

✅ Predict Dissolved Oxygen (DO)  
✅ Classify water quality (SAFE / WARNING / UNSAFE)  
✅ Interactive and user-friendly UI  
✅ Real-time prediction using Streamlit  
✅ Trained using Random Forest Algorithm  

---

## 🧠 Machine Learning Model

- Algorithm: **Random Forest Regressor**
- Input Features:
  - pH
  - Turbidity
  - Temperature
- Output:
  - Predicted DO (mg/L)
  - Water Quality Status

Random Forest is an ensemble learning method that improves prediction accuracy and reduces overfitting. :contentReference[oaicite:1]{index=1}
